# Web Portfolio

Portfolio statis (single-page) untuk Arief Hidayat Sutomo.

## Jalankan lokal

- Cara paling cepat: buka `index.html` langsung di browser.
- Atau via local server:

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Lalu buka `http://127.0.0.1:8000/`.

## Publish (GitHub Pages)

Repo ini bisa dipublish sebagai site statis via GitHub Pages (branch `main` / folder root).

## CV Download

Tombol Download CV mengarah ke file berikut (taruh di root repo, sejajar dengan `index.html`):

- `Arief-Hidayat-Sutomo-CV.pdf`
